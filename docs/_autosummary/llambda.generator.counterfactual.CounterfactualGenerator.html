class llambda.generator.counterfactual.CounterfactualGenerator(model_name=None, langchain_llm=None, temperature=1, max_calls_per_min=None, **kwargs)#

Bases: ResponseGenerator

__init__(model_name=None, langchain_llm=None, temperature=1, max_calls_per_min=None, **kwargs)#

Class for parsing and replacing protected attribute words.

For the full list of gender and race words, refer to https://github.aetna.com/analytics-org/llambda/blob/main/llambda/bias_word_lists.py

Parameters:

  • model_name (str, default=None) – Specifies which model deployment to use with langchain. If using a GPT model, specifies deployment_name parameter for AzureChatOpenAI (langchain) or model for openai.chat.completions.create(). If using a Gemini or PaLM/bison model, specifies model name. Must be provided if langchain_llm is None. Currently, only GPT, Gemini and PaLM/bison models are supported.

  • langchain_llm (langchain llm object, default=None) – A langchain llm object to get passed to LLMChain llm argument. Must be provided if model_name is None. User is responsible for specifying temperature and other relevant parameters to the constructor of their langchain_llm object.

  • temperature (float between 0 and 2, default=1) – Specifies what sampling temperature to use. Only used if langchain_llm is None. Temperature must be greater than 0 to use count > 1 with generate_responses method.

  • max_calls_per_min (int, default=None) – Specifies how many api calls to make per minute to avoid a rate limit error. By default, no limit is specified.

  • **kwargs (Keyword arguments.) – These will be passed to the constructor of AzureChatOpenAI, VertexAI, or ChatVertexAI, depending on the model_name specified.

async estimate_token_cost(tiktoken_model_name, prompts, attribute, example_responses=None, response_sample_size=30, system_prompt='You are a helpful assistant', count=25)#

Estimates the token cost for a given list of prompts and (optionally) example responses. Note: This method is only compatible with GPT models.

Parameters:

  • prompts (list of strings) – A list of prompts

  • tiktoken_model_name (str) – The name of the OpenAI model to use for token counting.

  • attribute (str, either 'gender' or 'race') – Specifies attribute to be used for counterfactual generation

  • example_responses (list of strings, default=None) – A list of example responses. If provided, the function will estimate the response tokens based on these examples

  • response_sample_size (int, default = 30.) – The number of responses to generate for cost estimation if example_responses is not provided.

  • system_prompt (str, default="You are a helpful assistant.") – The system prompt to use.

  • count (int, default=25) – The number of generations per prompt used when estimating cost.

Return type:

Dict[str, float]

Returns:

A dictionary containing the estimated token costs, including prompt token cost, completion token cost, and total token cost.

Creates prompts by counterfactual substitution and generates responses asynchronously
